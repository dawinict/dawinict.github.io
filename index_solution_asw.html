ASW (AppSmartWorker) 솔루션

What is ASW

ASW(AppSmartWorker)는 엔터프라이즈 환경에서 수행되는 애플리케이션 마이그레이션을 대상으로 소스 분석부터 자동 전환, 이력 관리까지 전 과정을 자동화하는 애플리케이션 전환 자동화 솔루션입니다.



ASW 자동화 플랫폼 아키텍처

애플리케이션 전환 자동화 솔루션




기존 전환 방식의 한계  ● 개발자 숙련도에 따른 품질 편차  ● 반복 작업으로 인한 오류 발생  ● 대규모 전환 시 일정 통제 어려움  ● 전환 결과 관리의 한계
ASW의 접근 방식  ● 전환 로직의 Rule 기반 표준화  ● 자동 전환 엔진을 통한 일관된 품질 확보  ● 전환 전·후 결과의 체계적 관리

ASW는 AS-IS 애플리케이션을 분석한 후 Rule 기반 자동 전환 엔진을 통해 TO-BE 환경으로 전환하며, 전환 결과를 중앙에서 관리합니다.

 ● 전환 대상 소스 자동 분석 및 Assessment  ● Rule 기반 소스 자동 변환  ● 전환 전·후 소스 비교 및 이력 관리  ● 반복 전환 프로젝트 표준화

 ● Unix → Linux(U2L) 전환  ● 노후 플랫폼 현대화  ● 대규모 애플리케이션 일괄 전환  ● 반복 전환 프로젝트 표준화

 ● 전환 기간 단축  ● 전환 품질 향상 및 오류 감소  ● 대규모 전환 프로젝트 일정·품질 통제

ASW는 엔터프라이즈 IT 전환을 위한 애플리케이션 전환 자동화 전문 솔루션입니다.
